# Not a Number

## Project idea
This project is about eating disorder thoughts, food anxiety, and body image pressure.  
The main idea is that when someone only sees food and body through numbers, their mind gets louder and more painful.  
The page compares that with a calmer mode focused on trust and nourishment.

## What the page does
- Starts on an enter screen.
- Lets users switch between **Numbers Mode** and **Peace Mode**.
- Shows clickable bubbles around the character.
- Changes the character pose and expression based on clicks.
- Adds floating words as emotional/noise feedback.
- Shows ending messages after enough clicks in each mode.

## Files
- `index.html` - page structure
- `style.css` - visual styles and animations
- `script.js` - interactions and state updates

## How to run
- Open `index.html` directly in a browser, or
- Open the GitHub Pages link for this repo.

## Reflection
This project helped me practice DOM interaction, CSS animation, and using visual design to communicate an emotional idea in a simple single-page website.
